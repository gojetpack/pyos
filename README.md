#### pyos

Golang package which contains features similar to the python os module

Current support:

##### Modules

| Python  | Golang |
| ------- |:------:|
| os.name | pyos.Name |
| os.path.exist() | pyos.Path.Exist() |
| os.path.isfile() | pyos.Path.IsFile() |
| os.path.isdir() | pyos.Path.IsDir() |



Note: Development in progress
